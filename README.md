# Ngao Sports Website

A modern, responsive sports website featuring various activities including Football, Basketball, Skating, Robotics, Chess, and Music.

## Features

- **Responsive Design**: Works on all screen sizes from mobile to desktop
- **Interactive Animations**: Scroll-triggered zoom animations for engaging user experience
- **Dropdown Navigation**: Events dropdown (Upcoming/Past events) and Activities dropdown (all sports)
- **Contact Form**: Email form that sends messages to james.o.nderema@gmail.com
- **Multiple Pages**: Home, About, Events, Contact, and individual Activity pages

## Pages

- `index.html` - Home page with hero section, sports cards, stats, and upcoming events
- `about.html` - About page with mission, vision, values, and team sections
- `events.html` - Events page with upcoming and past events
- `contact.html` - Contact page with form and FAQ section
- `activities/` - Individual pages for each sport:
  - `football.html`
  - `basketball.html`
  - `skating.html`
  - `robotics.html`
  - `chess.html`
  - `music.html`

## Technologies Used

- HTML5
- CSS3 (with responsive media queries and animations)
- Vanilla JavaScript (for interactivity and scroll animations)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Or serve with any web server:
   ```bash
   python -m http.server 8080
   ```
